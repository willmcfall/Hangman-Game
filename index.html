Welcome to the Ricky and Morty Hangman Game!



Instructions

This game is a unique and modern take on the classic children's game Hangman. Users are asked to guess the name of prominent characters from the hit television show Rick and Morty through an iterative process of guessing individual letters from characters name and reflecting on newly shown letters. If the user can guess all the letters of the character's name before providing 10 wrong guesses, they have won the round and are provided with an additional character name. Please press the start button below to begin the game.
